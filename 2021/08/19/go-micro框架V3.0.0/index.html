安装

安装go-micro框架

需要用到Micro 3.0 环境的micro 工具(二进制可执行文件)，可以快速构建项目。但是在代码中不使用该库。

首先看一下帮助

创建一个新服务

micro new serviceName

默认在__当前路径__下生成对应服务的文件

看一下proto文件

使用make proto编译proto文件

就这样非常简单的生成了一个服务microHello。

需要注意的是：这里使用go-micro框架，于是注释掉了自动生成的micro代码, 并且使用go-micro 插件consul

同样地，将proto文件复制到客户端，编写客户端代码。

测试

启动consul服务器

简单测试，直接使用命令consul agent -dev来启动一个consul服务。

rpc_server注册服务(microHello)

rpc_client调用服务

mongoDB安装

安装步骤链接

systemctl start mongod启动mongoDB，默认监听127.0.0.1:27017

连接数据库

show databases和mysql一样展示所有数据库。

mongoDB不需要创建database

use db1和mysql一样使用db1数据库。需要注意的是，mongoDB这里的数据库只是一个命名空间而已。

db.createCollection("table1")创建表，由于是非关系型数据库，建表时并不需要指定表结构。

show collections展示所有表

db.table1.drop()删除表

db.table1.insert()插入一个(返回WriteResult对象)或者多个(返回BulkWriteResult对象)

db.table1.insertOne({})插入一个

db.insertMany([{},...,{}])插入多个

db.table1.find()查询表，相当于select * from table1;

db.table1.find({name: "gys"})查询表，相当于select * from table1 where name = "gys";

db.table1.remove({name: "gys"})删除记录，相当于delete from table1 where name = "gys";

db.table1.createIndex({uid:-1})创建索引, 1代表正向索引，-1代表反向索引

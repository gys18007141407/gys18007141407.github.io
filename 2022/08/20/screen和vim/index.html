screen和vim

screen

  • 配置文件

screen -S xx：创建

screen -r xx：恢复

screen -ls：列表

ctrl a ctrl d：detach

exit：退出窗口

screen -S xx -X quit：关闭

screen -x xx：共享screen

ctrl a ctrl c：创建窗口

ctrl a shift s：分屏

ctrl a tab：切换分屏

ctrl a ctrl a：上一个窗口

ctrl a num：num号窗口

vim

  • 配置文件

.bashrc设置user@host风格

  • 打开双屏显示另一个文件（水平切割split，垂直切割vsplit）

vsp another/file

  • 窗口之间切换（只开两个窗口，多了太乱）

ctrl+w+w 循环切换窗口

ctrl+w+←/h 左边一个窗口

ctrl+w+→/l 右边一个窗口

ctrl+w+↑/k 上面一个窗口

ctrl+w+↓/j 下面一个窗口

  • 关闭所有窗口

qall

  • 只留下当前窗口，其他窗口关闭

only

  • 高亮显示当前光标所在单词

g+d

n 跳到下方一个高亮的位置

N 跳到上方一个高亮的位置

  • 代码跳转

ctrl+] 跳转到函数定义处

ctrl+o 跳转到函数调用处

git

1、远程创建仓库

2、本地初始化仓库(可能要先设置用户名邮箱啥的git config -l) git init

3、关联远端 git remote add origin git@github.com:账号名/仓库名.git

4、设置上游 git branch –set-upstream-to=origin/master

5、拉取远端 git pull –rebase （–rebase本地变基）

6、push本地
